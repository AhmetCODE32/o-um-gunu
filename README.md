# Doğum Günü Kutlaması

Bu proje, Altar için tamamen özel olarak tasarlanmış olan interaktif bir doğum günü sürprizi web sayfasıdır. 

## Proje Hakkında
- **Tasarım:** Siyah, Mor ve Cam (Glassmorphism) detaylı özel arayüz.
- **İçerik:** Özelleştirilmiş mesajlar, arkadaşlığa özel "Yazılı Olmayan Kurallar" manifesto alanı ve animasyonlu gizli hediye paketi.
- **Uyumluluk:** Hem telefonlarda hem de bilgisayarlarda kusursuz, kaydırılabilir görünüm (%100 Responsive).

## Yerelde Çalıştırmak İçin (Kurulum)

Projeyi kendi bilgisayarınızda çalıştırmak isterseniz aşağıdaki adımları izleyebilirsiniz:

1. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```

2. Geliştirici sunucusunu başlatın:
   ```bash
   npm run dev
   ```
